قلب المنصة: Escrow Logic

منطق العمل: Hold -> Verify -> Release لضمان حق العميل والتاجر.

  1. 1. عند الدفع، ينتقل المبلغ إلى محفظة حجز آمنة.
  2. 2. الطلب يظهر للتاجر بحالة "مدفوع - بانتظار الاستلام".
  3. 3. مسح Pickup Code يطلق عملية التحقق والتحرير.
  4. 4. معالجة النزاع متاحة قبل الإغلاق النهائي للحالة.
